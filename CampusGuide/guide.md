# QRコードの形式の例
{ "qr_id": 5, "name": "講堂", "x": 560, "y": 160, "floor": 1}  
注意点として、{}で囲むのと、「,」で区切ること、""で囲むこと、間に「:」を入れることは忘れないでください。要は各部分の値だけ変えればいいってことです。
